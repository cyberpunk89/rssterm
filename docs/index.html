Features

⌂ Home tab
Combined feed from all categories, deduplicated and sorted newest first.
🖼 Inline images
Preview pane renders article images using terminal-image — pure JS, no external binaries.
⚙ In-app config
Add, edit, or remove feeds and categories without touching YAML. Press c.
▶ YouTube RSS
Paste any YouTube channel URL — auto-converts to the correct RSS feed URL.
/ Search
Live filter across the active tab by title or source name. Press /.
⌨ Vim navigation
jkhl + arrow keys. Tabs loop around at both ends.
󰖟 ASCII art headers
Each category tab has a unique ASCII art block. Keyed by name or icon character.
⚡ Cache-first
Loads stale content instantly on startup while refreshing in the background.

Requirements

  • Node.js ≥ 20
  • pnpm

Installation

bash
git clone <repo-url> rssterm
cd rssterm
pnpm install

# Create config directory and copy the example config
mkdir -p ~/.config/rssterm
cp config.example.yaml ~/.config/rssterm/config.yaml

# Edit the config to add your feeds
$EDITOR ~/.config/rssterm/config.yaml

# Run
pnpm start

Optional shell alias (fish) — add to ~/.config/fish/config.fish:

fish
alias rss="pnpm --prefix ~/path/to/rssterm start"

Configuration

Config file: ~/.config/rssterm/config.yaml

yaml
refreshOnStart: true      # fetch all feeds on startup

categories:
  - name: Tech
    icon: "󰖟"             # any character — Nerd Font glyph, emoji, ASCII
    feeds:
      - name: The Verge
        url: https://www.theverge.com/rss/index.xml
      - name: Hacker News
        url: https://news.ycombinator.com/rss

  - name: Gaming
    icon: "󰊗"
    feeds:
      - name: IGN
        url: https://feeds.feedburner.com/ign/all

You can also manage feeds from inside the app — press c to open the config panel.

YouTube channels

Paste any YouTube channel URL when adding a feed — rssterm converts it automatically:

InputConverted to
youtube.com/channel/UCxxxxxxxfeeds/videos.xml?channel_id=UCxxxxxxx
youtube.com/@handlefeeds/videos.xml?channel_id=@handle
youtube.com/c/namefeeds/videos.xml?user=name

When adding a feed URL, YouTube channel URLs are detected and a preview of the converted RSS URL is shown inline. When creating a category, an icon picker lets you choose using ← / →.

Customization

Theme

Colors are defined in src/lib/terminal-themes/catppuccin.ts. Change any hex value and restart.

ASCII art headers

src/lib/ascii-art.ts contains a lookup table keyed by category name and icon character. Add entries to give any category its own art.

Article list height

typescript
// src/components/ArticleList.tsx
const LIST_HEIGHT = 10; // visible rows

Adding a keybinding

  1. Add a branch inside the useInput handler in src/app.tsx
  2. Add it to KEYBINDS in src/components/HelpOverlay.tsx
